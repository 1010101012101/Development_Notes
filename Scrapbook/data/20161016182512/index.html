ATmega16U4
device
ATMEGA16U4-AUR |ATMEGA16U4-MU |ATMEGA16U4-AU |ATMEGA16U4-MUR

The low power, high performance Atmel 8-bit AVR RISC-based microcontroller featuring 16KB ISP flash memory, 8KB boot code section, 2.5KB SRAM, 1024-byte EEPROM, USB 2.0 full-speed/low speed device, high-speed PWMs, 12-channel 10-bit A/D-converter, and JTAG interface for on-chip-debug. The device achieves up to 16 MIPS throughput at 16 MHz. 2.7 - 5.5V operation.

By executing powerful instructions in a single clock cycle, the device achieves throughputs approaching 1 MIPS per MHz, allowing you to optimize power consumption versus processing speed.
