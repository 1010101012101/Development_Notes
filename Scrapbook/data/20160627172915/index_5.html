                                                                                                                                              That is correct, and one way to do it. A better way would be to pass Popen a list, like this,

                                                                                                                                              proc = Popen(['svn', 'co', 'svn+ssh://myrepo'], stdout=PIPE, stderr=PIPE)

                                                                                                                                              This is described in the "Running via the shell" section. Basically when you passable string, it will be interpreted by the shell if she'll is True, else python will interpret it. When python does it, it looks for an executable file with the whole string as the name, which obviously is not found. So, both these solutions are correct, but passing a list might be slightly faster as it doesn't start a shell.
